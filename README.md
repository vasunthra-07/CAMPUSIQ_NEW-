## Overview

CampusIQ is an intelligent campus operating system designed to unify academic, administrative, operational, and student services into a single platform.

Educational institutions often rely on multiple disconnected systems for attendance, events, communication, resources, transport, service requests, and campus operations.

CampusIQ brings everything together into one centralized operational workspace powered by automation, analytics, and AI.

---

## Problem

Modern campuses face challenges such as:

- Fragmented management systems
- Resource underutilization
- Poor operational visibility
- Delayed communication
- Inefficient complaint resolution
- Lack of real-time decision support

These issues lead to reduced efficiency and poor stakeholder experiences.

---

## Solution

CampusIQ acts as a digital operating layer for institutions by providing:

- Unified Campus Operations
- Smart Resource Management
- Event Lifecycle Management

- AI-Powered Assistance
- Communication & Alert Systems
- Operational Intelligence Dashboard

# Getting Started

## Clone Repository

```bash
git clone https://github.com/kiyotakaaKira/CAMPUS-IQ.git
cd CAMPUS-IQ
```

## Install Frontend Dependencies

```bash
npm install
```

## Install Backend Dependencies

```bash
cd backend
npm install
```

## Configure Environment Variables

Create:

```env
backend/.env
```

Example:

```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=3001
OLLAMA_URL=http://localhost:11434
```

## Run Backend

```bash
cd backend
npm start
```

Backend:

```text
http://localhost:3001
```

## Run Frontend

```bash
npm run dev
```

Frontend:

```text
http://localhost:5173
```
