# How to Run CIT-Sentinel

## Terminal 1 — Backend
cd backend
npm install
npm run dev

## Terminal 2 — Ollama AI (optional)
ollama serve

## Terminal 3 — Frontend
npm run dev

## Access
Open http://localhost:8080 in browser
